# enfrm-app

#### Description
entfrm-app是基于uni-app 框架、使用 Vue.js 语法开发的移动端 App开源产品。它可以编译为 H5、IOS App、Android App、微信小程序、QQ小程序、钉钉小程序、支付宝小程序、百度小程序、字节跳动小程序，总共9个运行平台的开源产品，而只需编写一套代码。
